## Syntax

`@since <versionDescription>`


## Overview

The @since tag indicates that a class, method, or other symbol was added in a specific version.


## Examples

- **Using the @since tag**

```js
/**
 * Provides access to user information.
 * @since 1.0.1
 */
function UserRecord() {}
```
